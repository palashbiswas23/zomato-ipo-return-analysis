Project Title
Zomato IPO Return Analysis Dashboard

Project Overview
This project analyzes the investment performance of Zomato since its IPO using financial data and stock price data.
The objective is to understand how much return an investor would have earned if they invested at the IPO price and held the stock over different time horizons.

Company Analyzed

| ------------- | ------------ |
| IPO Date      | 23 July 2021 |
| Issue Price   | ₹76          |
| Listing Price | ₹116         |

Tools Used
Excel – Data cleaning and return calculations
Power BI – Interactive dashboard visualization
GitHub – Project documentation and portfolio

Dashboard Features
KPI cards (IPO price, current price, total return)
Stock price trend since IPO
Investor return trend
Trading volume analysis
Holding period return comparison

Key Insights
• Zomato delivered strong listing gains for IPO investors.

• The stock experienced volatility during the first year after listing.

• Long-term investors benefited from price appreciation after initial fluctuations.

• Trading volume spikes corresponded with major price movements.
